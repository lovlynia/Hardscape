# Hardscape 🌿
A website created for Phoenix Stoneworks LLC to archive it's work 

 Pages<br>
index.html — Home<br>

Full-screen hero with a background castle/stone image<br>
Services overview with logo, service list, and equipment rentals<br>
Photo grid showcasing past work (patios, walkways, steps, pools)<br>
Call-to-action buttons linking to the gallery and contact page<br>

about.html — About

Split-screen layout: animated Spline 3D viewer on the left, photo on the right<br>
Fully responsive — the photo panel hides on mobile<br>

gallery.html — Our Work<br>

Masonry-style CSS column grid auto-populated from a JavaScript data array<br>
Click any project card to open a lightbox carousel<br>
Carousel supports: arrow buttons, dot indicators, keyboard navigation (← → Esc), and click-outside-to-close<br>

contact.html — Contact<br>

Full-page form over a blurred background image<br>
Form submission powered by Web3Forms (no backend needed)<br>
Fields: Name, Email, Message<br>


✨ Features<br>

Responsive design — mobile hamburger sidebar navigation<br>
Lightbox gallery — per-project image carousels with keyboard support<br>
Spline 3D embed — interactive 3D animation on the About page<br>
Web3Forms integration — serverless contact form handling<br>
CSS custom properties — consistent theming via :root variables<br>
Google Fonts — Playfair Display + Crimson Text for an editorial, craft-forward feel<br>

--Getting Started<br>

```
git clone https://github.com/your-username/phoenix-stoneworks.git
cd phoenix-stoneworks
```
Open index.html in your browser, or serve it locally:
```
# Python
python -m http.server 8000

# Node (npx)
npx serve .
```
