# Hardscape 🌿
A website created for Phoenix Stoneworks LLC to archive it's work 

The website has 3 pages , it's home page, about page and gallery page 
The work tab also contains a link section in the home page demonstrating the companies best work 

The first page is used to draw attention the the companies services and their best work. 

The web page also contains a footer available on each instance of the website 
with contact us referring to the company's business email. 

The website contains interactive features such as it's sidebar 
The carousel in the gallery page and about page that are both being updated - removing old projects that can be see through the companies preferred platform Facebook 

This website uses html, css, and light js along with interactive 3D design using Spline
